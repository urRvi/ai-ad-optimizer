# AI‑Driven Ad Copy & Landing Page Optimizer
Generate high‑converting ad headlines, descriptions, CTAs and landing‑page sections from a brief.

## Quickstart (MVP)
1) `pip install streamlit openai python-dotenv pandas`
2) Set `OPENAI_API_KEY` in `.env`
3) `streamlit run mvp_streamlit/app.py`

## Stack
Python, Streamlit/FastAPI, SQLite, React (optional), OpenAI API
